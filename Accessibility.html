Accessibility Statement

We are committed to making the University of Sheffield Guide accessible to all users, including those with disabilities. Our goal is to follow the Web Content Accessibility Guidelines (WCAG) 2.1 to ensure the website is usable for everyone, regardless of the device or assistive technology used.

What We Have Done

  • Visual Accessibility: High-contrast text and background color combinations ensure readability for users with visual impairments. Alternative text (alt text) has been added for all images to describe them for screen reader users.
  • Keyboard Navigation: All interactive elements, such as buttons, links, and forms, are focusable and navigable via keyboard, ensuring usability for users with motor impairments.
  • Screen Reader Compatibility: We have used proper ARIA roles and properties, along with semantic HTML tags, to make the website compatible with screen readers such as JAWS, NVDA, and VoiceOver.
  • Forms and Buttons: Forms and buttons have clear labels, making them accessible for all users, including those using assistive technologies.

Reasoning for These Choices

These decisions were made to meet the WCAG 2.1 standards:

  • High contrast improves visibility (WCAG 1.4.3).
  • Alt text for images supports users who rely on screen readers (WCAG 1.1.1).
  • Keyboard navigation ensures the site is accessible to users with motor impairments (WCAG 2.1.1).

Issues Handled by Other Software

Some accessibility issues are handled by external software:

  • Screen Readers: Tools like JAWS, NVDA, and VoiceOver help blind or visually impaired users navigate the site.

We recommend using these tools for a better browsing experience. Additionally, browser extensions like Grammarly and Read&Write can enhance text-to-speech functionality and improve readability.
